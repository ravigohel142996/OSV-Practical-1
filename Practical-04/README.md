# Practical 4: Menu-Driven Programs and Case Statements

## Aim
To write shell scripts for:
a) Menu-driven program for file operations (copy, move, delete)
b) Menu-driven calculator using case statement
c) Printing days of the week

## Programs Included
1. `file_operations.sh` - File operations menu
2. `calculator.sh` - Menu-driven calculator
3. `weekdays.sh` - Display days of the week

## Commands to Run
```bash
chmod +x *.sh
./file_operations.sh
./calculator.sh
./weekdays.sh
```

## Viva Questions
1. What is a case statement in bash?
2. How does case statement differ from if-else?
3. What are file operation commands in Linux?
4. How do you create a menu-driven program?
