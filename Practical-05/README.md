# Practical 5: Loops in Shell Scripting

## Aim
To write shell scripts for:
a) Finding factorial of a number
b) Printing natural numbers from 1 to n
c) Printing even and odd numbers using while loop

## Programs Included
1. `factorial.sh` - Calculate factorial
2. `natural_numbers.sh` - Print natural numbers
3. `even_odd.sh` - Print even and odd numbers

## Commands to Run
```bash
chmod +x *.sh
./factorial.sh
./natural_numbers.sh
./even_odd.sh
```

## Viva Questions
1. What are different types of loops in bash?
2. How does while loop differ from for loop?
3. What is factorial of a number?
4. How do you increment a variable in bash?
