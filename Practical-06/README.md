# Practical 6: Number and String Manipulation

## Aim
To write shell scripts for:
a) Reversing a number
b) Checking if a number is prime
c) Checking if a string or number is palindrome

## Programs Included
1. `reverse_number.sh` - Reverse a number
2. `prime_check.sh` - Check if number is prime
3. `palindrome.sh` - Check palindrome

## Commands to Run
```bash
chmod +x *.sh
./reverse_number.sh
./prime_check.sh
./palindrome.sh
```

## Viva Questions
1. What is a palindrome?
2. What is a prime number?
3. How do you reverse a number?
4. What is the modulus operator used for?
