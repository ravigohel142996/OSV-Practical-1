# Practical 7: Advanced Shell Scripting

## Aim
To write shell scripts for:
a) Generating first n Fibonacci numbers
b) Converting lowercase to uppercase and vice versa using awk

## Programs Included
1. `fibonacci.sh` - Generate Fibonacci series
2. `case_converter.sh` - Convert case using awk

## Commands to Run
```bash
chmod +x *.sh
./fibonacci.sh
./case_converter.sh
```

## Viva Questions
1. What is Fibonacci series?
2. What is awk command used for?
3. How does awk process text?
4. What are some common uses of awk?
