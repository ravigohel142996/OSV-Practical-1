# Practical 2: Arithmetic Operations and Comparisons

## Aim
To write shell scripts for:
a) Arithmetic operations on two numbers
b) Finding greater of two numbers
c) Finding smallest and largest of three numbers

## Programs Included
1. `arithmetic.sh` - Perform arithmetic operations
2. `greater.sh` - Find greater of two numbers
3. `smallest_largest.sh` - Find smallest and largest of three numbers

## Commands to Run
```bash
chmod +x *.sh
./arithmetic.sh
./greater.sh
./smallest_largest.sh
```

## Viva Questions
1. How do you perform arithmetic operations in bash?
2. What is the difference between [] and (()) for comparisons?
3. How does the if-else statement work in shell scripts?
4. What are relational operators in bash?
