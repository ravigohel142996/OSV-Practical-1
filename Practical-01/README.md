# Practical 1: Introduction to Shell Scripting

## Aim
To write shell scripts for:
a) Printing a welcome message
b) Accepting and displaying details of two students

## Theory
Shell scripting is a powerful way to automate tasks in Linux. A shell script is a file containing a series of commands that are executed sequentially by the shell interpreter (bash).

## Programs Included
1. `welcome.sh` - Print welcome message
2. `student_details.sh` - Accept and display student details

## Commands to Run
```bash
# Make scripts executable
chmod +x welcome.sh
chmod +x student_details.sh

# Run welcome script
./welcome.sh

# Run student details script
./student_details.sh
```
